GoChar

goChar es un proyecto de código libre basado en el reconocimiento de caracteres óptico distribuido, mediante el uso de las llamadas a procedimientos remotos(RPC). Está creado a partir del lenguaje de programación Go, Python y la librería open-source TensorFlow.

El reconocimiento de caracteres óptico es un proceso dirigido a la digitaliación de textos, los cuales se identifican a partir de una imagen de simbolos o caracteres que pertenecen a un determinado alfabeto.

Muchos lientes desde distintas plataformas pueden acceder a este sistema y subir una imagen para digitalizar una imagen.

Go es un lenguaje de programación inspirado en la sintaxis de C. Nacio en 2009 creado por Google, algunos de sus creadores son los padres de Unix, UTF-9 y V8-Javascript Engine. Caracteristicas:

  • Compilado
  • Estaticamente tipado
  • Concurrente
  • Uso atípico de Programacion O. O. - Go no usa clases, no usa herencia y el uso de interfaces se realiza de manera implicita.
  • Todo esto pensado para mejorar el rendimiento al momento de diseñar el software.

Go está basado en paquetes, realizados por una amplia comunidad de desarrolladores de todo el mundo. Para este proyecto hemos utiliado el paquete RPC2 de golang, desarrollado por cenk. Este paquete es un fork de la paquete de golang net/rpc, pero el cual añade comunicación bidireccional, esto significa que el servidor puede llamar a los metodos del cliente.

Cada nodo ejecuta un script en Python para el reconocimiento de los caracteres. Este programa toma una imagen en 28x28 en escala de grises, la salida de este es el carácter que ha obtenido. El reconocimiento de caracteres está basado en un conjunto de redes neuronales y convolucionales. Tras seguir por estas capas, se somete a la imagen a una regresión logistica para determinar el caracter del que se trata. El script te Python trabaja con TensorFlow, creado por Google. Es una librería de código libre para el aprendizaje de máquinas.
