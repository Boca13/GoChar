GoChar

GoChar es un proyecto de código libre basado en el reconocimiento de caracteres óptico distribuido, mediante el uso de las llamadas a procedimientos remotos(RPC). Está creado a partir del lenguaje de programación Go, Python y la librería open-source TensorFlow.

El reconocimiento de caracteres óptico es un proceso dirigido a la digitaliación de textos, los cuales se identifican a partir de una imagen de simbolos o caracteres que pertenecen a un determinado alfabeto.

Muchos clientes desde distintas plataformas pueden acceder a este sistema y subir una imagen para digitalizar una imagen.

Estructura del proyecto

Front end

  • Sirve la página web en la que estás navegando ahora mismo

Back end - Maestro RPC

  • Contiene al mismo tiempo un servidor HTTP y un servidor RPC
  • Completamente programado en Go
  • Protocolo HTTP -> usuarios ; RPC -> nodos

  • Métodos que exporta servidor HTTP
    • Subir. Admite una imagen por HTTP POST. Responde con un id de trabajo asociado al cliente
    • Estado: Admite un id de trabajo del cliente, devuelve el caracter resultante

  • Métodos que exporta servidor RPC
    • AceptaConexiones: abre una conex. con el nodo y guarda info acerca de el en lista enlazada
    • CierraConexiones: Recibe un id de nodo y elimina al nodo de la lista enlazada. Tambien cierra la conex.
    • RecibeRespuesta: Admite un byte con el caracter del resultado. Actualia el nodo en la lista

Esclavo RPC

  • Encargado de conectarse al maestro para proporcionar servicio OCR
  • Completamente programado en Go.

  • Métodos RPC que exporta
    • RecibeImagen: Recibe un tipo imagen, exporta esta a un fichero y llama scrip Python para reconocer el caracter. Cuando termina hace un callback y llama a RecibeRespuesta
