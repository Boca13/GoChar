GoChar

GoChar es un proyecto de código libre basado en el reconocimiento de caracteres óptico distribuido, mediante el uso de las llamadas a procedimientos remotos(RPC). Está creado a partir del lenguaje de programación Go, Python y la librería open-source TensorFlow.

El reconocimiento de caracteres óptico es un proceso dirigido a la digitaliación de textos, los cuales se identifican a partir de una imagen de simbolos o caracteres que pertenecen a un determinado alfabeto.

Muchos lientes desde distintas plataformas pueden acceder a este sistema y subir una imagen para digitalizar una imagen.

Go

Go es un lenguaje de programación inspirado en la sintaxis de C.

Creado por Google, 2009. Algunos de sus creadores son los padres de Unix, UTF-9 y V8-Javascript Engine.

Caracteristicas:

  • Estaticamente tipado
  • Concurrente
  • Uso atípico de Programacion O. O. - Go no usa clases, no usa herencia y el uso de interfaces se realiza de manera implicita.

Go está basado en paquetes, realizados la mayor parte por la comunidad.
